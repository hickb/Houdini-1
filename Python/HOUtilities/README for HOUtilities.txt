########################################
# README for HOUtilities
########################################

HOUtilities are few small but handy utilities to help Houdini artists/TDs with their workflow.


########################################
Installation :
########################################
1. Copy HOUtilities folder to [your Houdini 14 installation]\houdini\python2.7libs\
	e.g. C:\Program Files\Side Effects Software\Houdini 14.0.201.13\houdini\python2.7libs\HOUtilities
2. Restart Houdini
3. Create pypanels with the code provided in .pypanel files
